INSTALLATION PROCESS:

Step 1:

Reboot into TWRP [twrp_3.6.0_11-RedmiK30Pro_v3.3_A12-lmi-skkk_e185dab6.img]
Flash the "systemrw_1.32_flashable" zip (To make the system "RW" Readable and Writable)
Reboot the TWRP 
Flash "Magisk-v24.0.apk", using OTG
Reboot the device, open Magisk app then reboot the device to finish the Magisk installation
Reboot the TWRP again
Go to "Mount" and mount: the "Vendor", "System" and "System_ext" partitions


Step 2: Open a new CMD or a Terminal and type in the following commands to mount some directories for adb shell:

adb shell
mount -o rw,remount /
mount -o rw,remount /vendor
mount -o rw,remount /system_ext

Cool tips:
How to grant su permission to run any code with adb (for USB Debugging):
adb shell su -c ls <your_code> 
.............

How to Delete files from root directories:
The device must be rooted!
# Go to "root-explorer" folder and install the following apks:
es-file-explorer-4-2-9-2-1.apk
BusyBox-6.8.2(68003).apk

Open BusyBox and grant it root permission! 
Open Es File Explorer and close it, reopen it again, and grant it root permission. 
Then, go Tools option and enable "Root Explorer" then choose "Mount R/W" option.
.............


How to pull system and system_ext folders via adb from root directory (using USB Debugging):
adb shell su -c mount -o rw,remount /
adb pull system
adb pull system_ext
.............
